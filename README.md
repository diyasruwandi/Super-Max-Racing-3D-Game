## 🎮 Features
- Formula 1 style racing gameplay
- Qualification Mode with best lap time system
- Race Mode with AI opponents
- Wet weather racing environment
- Realistic car handling using Wheel Colliders
- Lap counter and lap timer system
- Race position system
- Finish screen with winner display
- Cockpit camera and third-person camera
- Camera switching system
- Engine sound system (idle, acceleration, deceleration)
- Restart and main menu system
- AI racing line system
- Cinemachine camera system
- Mobile joystick support

## 🛠 Technologies Used
- Unity Engine
- C#
- Cinemachine
- Unity Wheel Collider Physics
- Unity UI System
- TextMeshPro
- Unity Video Player
- Unity Audio System
- Unity Version Control / GitHub

## 🎮 How to Play
Qualification Mode
1. Start the qualification session from the main menu.
2. Drive around the track and complete laps.
3. Try to achieve the fastest lap time.
4. Control the car carefully through corners to maintain speed.
Race Mode
1. Start the race session.
2. Compete against AI opponents in wet track conditions.
3. Complete all laps before the AI.
4. Maintain your racing position and avoid losing control on slippery corners.
5. Reach the finish line to win the race.
Controls
PC Controls
- W / Up Arrow → Accelerate
- S / Down Arrow → Brake / Reverse
- A / Left Arrow → Turn Left
- D / Right Arrow → Turn Right
- Space → Brake
- C → Switch Camera
Mobile Controls
- Use the on-screen joystick to steer and accelerate the car.
